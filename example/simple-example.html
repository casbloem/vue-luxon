vue-luxon

Examples

The variable datetimeVariable is set to

{{ datetimeVariable | luxon }} <span v-luxon="datetimeVariable"></span>

{{ datetimeVariable | luxon }}

Options

{
value:' ',
serverZone:' ',
serverFormat:' ',
clientZone:' ',
clientFormat:' ',
localeLang:' ',

}

<span v-luxon="datetimeObject"></span> {{ datetimeObject | luxon }}

{{ datetimeVariable | luxon:locale }}

<span v-luxon:{{name}}="datetimeVariable"></span> {{ datetimeVariable | luxon:{{name}} }}
